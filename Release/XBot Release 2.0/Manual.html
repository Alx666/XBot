XBot Console Functions

setAlpha(alphaVal, TYPE);

=> alphaVal = alpha value(0-100)
=> TYPE = must be one of [MENU, MENU_FONT, CONSOLE, RADAR_PIX, RADAR_FONT, CROSSHAIR];

DEFAULT VALUE: 80 for all TYPE

Set the alpha value of the selected TYPE.

backgroundMode();

Set the console to run in background mode. In this mode console don't get keypress and don't slideIn after timeout.

connect();

Connect XBot to all the known IPs.

disconnect();

Disconnect XBot from all the known IPs.

clrScr();

Clear the console history.

setXHairSize(value, TYPE);

=> value = crosshair size(1-350);
=> TYPE = must be one of [ALL,SHIELD,ASSAULT_RIFLE,GES,SHOCK,LINK,MINIGUN,FLAK,ROCKET,SNIPER];

DEFAULT VALUE: 50 for all TYPE

Set the crosshair size for weapon TYPE.

setPlayerSpeedMode(MODE);

=> MODE = must be one of [NORMALIZED,NOT_NORMALIZED];

DEFAULT VALUE: NOT_NORMALIZED

Used in aim-algorithms computations, for low-framePerSecond PCs NORMALIZED is reccomended.

setAimSpeed(value,TYPE);

=> value = aimSpeed value (1-200);
=> TYPE = must be one of [ALL,SHIELD,ASSAULT_RIFLE,GES,SHOCK,LINK,MINIGUN,FLAK,ROCKET,SNIPER];

DEFAULT VALUE: 40 for all TYPE

Set the speed at which auto-aim moves your mouse for weapon TYPE.

setUnlockStrenght(value);

=> value = unlock str. value(1-100);

DEFAULT VALUE: 20

Set the amount of pixel to be moved with a single mouse movement to unlock current target.

enableTimer(value);

=> value = must be one of [ON, OFF];

DEFAULT VALUE: ON

Enable or disabel timers.

enableTimerSound(value);

=> value = must be one of [ON, OFF];

DEFAULT VALUE: ON

Enable or disabel timers sound played when time is critical.

addToStartup(str);

=> str = must be a valid function name with appropriate arguments, all enclosed within double quote;

Example: addToStartup( "setAlpha(80,CONSOLE) " )

Add the function specified between the 2 double quote to the startup.conf file.

setImodeType(value, TYPE);

=> value = must be one of [AUTO,FIXER,NONE_DISABLED,NONE_ENABLED];
=> TYPE = must be one of [ALL,SHIELD,ASSAULT_RIFLE,GES,SHOCK,LINK,MINIGUN,FLAK,ROCKET,SNIPER];

DEFAULT VALUE: NONE_DISABLED for all TYPE

NOTE: recommended settings are:

- shield gun : IMODE_TYPE_NONE_DISABLED;
- assault rifle : IMODE_TYPE_AUTO or IMODE_TYPE_FIXER;
- ges biorifle: IMODE_TYPE_AUTO or IMODE_TYPE_FIXER;
- shock rifle: IMODE_TYPE_AUTO or IMODE_TYPE_FIXER;
- link gun : IMODE_TYPE_AUTO or IMODE_TYPE_FIXER;
- minigun : IMODE_TYPE_AUTO or IMODE_TYPE_FIXER;
- flak cannon : IMODE_TYPE_FIXER;
- rocket launcher : IMODE_TYPE_FIXER;
- sniper rifle : IMODE_TYPE_AUTO or IMODE_TYPE_FIXER;
- lighting gun : IMODE_TYPE_AUTO or IMODE_TYPE_FIXER;

Set the ImodeType for weapon TYPE.

setWeaponRange(value, TYPE);

=> value = weapon range (0,10000) or [INFINITE];
=> TYPE = must be one of [ALL,SHIELD,ASSAULT_RIFLE,GES,SHOCK,LINK,MINIGUN,FLAK,ROCKET,SNIPER];

DEFAULT VALUE: INFINITE for all TYPE

Set the range in which XBOT will auto-aim your current target.

setAutoBlastComboRange(min, max);

=> min = must be a positive number;
=> max = must be a positive number;

DEFAULT VALUE: min = 140, max = 400

Set the range in which XBOT will auto-blast shock combos if they are near an enemy

join(roomNumber);

=> roomNumber = must be a positive number;

DEFAULT VALUE: the global room, Room n.0

Join the Room specified by roomNumber. When a player joins a room all the net traffic (except chat message) will be sent only to other players with the same room number.

leave();

Leave the current room. The current room will became the N. 0.

setTimeoutWeaponAutoSwap(value);

=> value = timeout value in millisec, must be a positive number;

DEFAULT VALUE: 3000

Set the idle timeout after an autoswapWeaponCombo is automatically stopped.

addAutoSwapWeaponCombo(fireMode, starterWeapon, .... , lastWeapon);

=> fireMode = must be one of [PRIM, SEC];
=> starterWeapon = must be one of [SHIELD,ASSAULT_RIFLE,GES,SHOCK,LINK,MINIGUN,FLAK,ROCKET,SNIPER,LIGHT_GUN];
=> ...... = you can insert 0 or more parameters here, each of them must be one of [SHIELD,ASSAULT_RIFLE,GES,SHOCK,LINK,MINIGUN,FLAK,ROCKET,SNIPER,LIGHT_GUN];
=> lastWeapon = must be one of [SHIELD,ASSAULT_RIFLE,GES,SHOCK,LINK,MINIGUN,FLAK,ROCKET,SNIPER,LIGHT_GUN];

Example: addAutoSwapWeaponCombo(PRIM, SHOCK, LIGHT_GUN, ROCKET, SHOCK)
Remember that at leat fireMOde,starterWeapon and lastWeapon parameters are required.

Add the SwapCombo you defined to the SwapCombo List.

getAutoSwapWeaponComboList();

Show the SwapCombo List.

clearAutoSwapWeaponComboList();

clears the SwapCombo List.

getPlayerList();

Show all the connected players and the rooms they are in.
